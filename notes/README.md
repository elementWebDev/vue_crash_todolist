## VUE-CLI 3

![vue cli 3](3-vue-cli3.png)

### CLI Commands

#### Install vue globally

```bash
npm -g @vue/cli
```

#### Check version installed

```bash
vue --version
```

#### Create Project

```bash
vue create
 - select defaults (babel, etc)

cd test
npm run serve

```

#### Vue GUI (dashboard, analytics, etc.)

Start GUI

```bash
vue ui
```

Stop Server

```bash
^C  // ctrl+c
```
